# Customer-Churn-and-Revenue-Loss-Analysis

## Project Overview
This project analyzes customer churn behavior using Power BI. It focuses on identifying high-risk customers, key churn drivers, and business patterns across demographics, subscriptions, and user activity.

---

## Objectives
- Identify customers likely to churn  
- Analyze churn patterns by region, age, and income  
- Understand impact of subscription type and contract  
- Find key drivers like login activity, complaints, and satisfaction  

---

## Dashboards Included

### 1. Overview Dashboard
- Total Customers, Churn Rate, Revenue  
- Churn by Subscription Type  
- Monthly churn trend  

### 2. Customer Behavior Dashboard
- Churn by Age Group, Gender, Region  
- High-risk segments  
- Contract type impact  

### 3. Churn Reason Dashboard
- Churn vs Satisfaction Score  
- Complaints & Login Activity analysis  
- Payment method & discount impact  

---

## Tools & Technologies
- Power BI  
- Power Query (Data Cleaning & Transformation)  
- DAX (Calculated Measures)  

---

## Key Insights
- Monthly subscription users have higher churn  
- Low satisfaction score strongly linked to churn  
- High complaints = higher churn probability  
- Certain regions show higher churn concentration  

---

## How to Use
1. Download `.pbix` file  
2. Open in Power BI Desktop  
3. Use slicers to explore different segments
